# Getting Started with React 19

React 19 brings exciting new features and improvements that will change how we build React applications. This comprehensive guide covers everything you need to know.

### Server Components by Default

React 19 makes Server Components the default, offering better performance out of the box:

```jsx
// This is now a Server Component by default
async function UserProfile({ userId }) {
  const user = await getUser(userId);
  return <div>{user.name}</div>;
}
```

### New use() Hook

The `use()` hook simplifies data fetching:

```jsx
import { use } from 'react';

function Post({ id }) {
  const post = use(fetchPost(id));
  return <h1>{post.title}</h1>;
}
```

### Enhanced Concurrent Features

Improved concurrent rendering with better user experience:

```jsx
import { startTransition } from 'react';

function updateProfile() {
  startTransition(() => {
    setProfile(newData);
  });
}
```

### Custom Hook Example

```jsx
import { useState, useEffect } from 'react';

function useLocalStorage(key, initialValue) {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      return initialValue;
    }
  });

  const setValue = (value) => {
    try {
      setStoredValue(value);
      window.localStorage.setItem(key, JSON.stringify(value));
    } catch (error) {
      console.error(error);
    }
  };

  return [storedValue, setValue];
}
```
